Slayd 8
Assets papka yaratish
Assets papka nima: Bu maxsus papka - bu yerda barcha rasmlarimizni saqlaymiz. Xuddi uy'dagi kiyim shkafi kabi.

Qayerda yaratish: Flutter loyihamizning asosiy joyida, "lib" papka yonida "assets" nomli papka yaratamiz.

Ichida qanday papkalar:
üìÅ assets/images/ - rasmlar uchun
üìÅ assets/icons/ - ikonlar uchun
üìÅ assets/fonts/ - harflar uchun

Birinchi qadam: "assets" papka yaratib, ichiga "images" papka qo'shamiz
Slayd 9
pubspec.yaml fayl bilan tanishish
pubspec.yaml nima: Bu Flutter loyihasining ID kartasi. Bu yerda loyiha haqida barcha ma'lumot yozilgan.

Nega kerak: Flutter'ga "men qaysi rasmlarni ishlatmoqchiman" deb aytish uchun bu faylga yozishimiz kerak.

Qayerda: Loyihaning eng asosiy joyida, "lib" papka yonida joylashgan.

Muhim qoida: Bu faylda bo'shliqlar juda muhim! Noto'g'ri bo'shliq qo'ysak, xatolik chiqadi.
Slayd 10
pubspec.yaml ga birinchi rasm qo'shish
Flutter'ga rasmni tanishtiramiz
assets/images/apple.png nomli rasm bor deb faraz qilamiz. pubspec.yaml faylini ochib, flutter qismini topamiz.
flutter: uses-material-design: true assets: - assets/images/apple.png
[pubspec.yaml fayli o'zgaradi - rasm ro'yxatga qo'shiladi]

Birinchi Assets rasm ko'rsatish
Image.asset() widget bilan
Image.asset() widget Assets papkadagi rasmni ekranda ko'rsatadi. Rasm yo'lini to'liq yozish kerak.
Center( child: Image.asset('assets/images/apple.png'), )
[Ekranda olma rasmi ko'rinadi]
Slayd 15
Rasm o'lchamini kichiklashtirish
width va height bilan
Rasm juda katta bo'lishi mumkin. width va height berib, kerakli o'lchamga keltirish mumkin.
Center( child: Image.asset( 'assets/images/apple.png', width: 100, height: 100, ), )
[100x100 o'lchamdagi olma rasmi]
Slayd 16
Bir nechta rasm yonma-yon
Row widget bilan
Row widget ichiga bir nechta Image.asset qo'yib, rasmlarni yonma-yon joylashtirish mumkin.
Center( child: Row( mainAxisAlignment: MainAxisAlignment.center, children: [ Image.asset('assets/images/apple.png', width: 80, height: 80), SizedBox(width: 20), Image.asset('assets/images/banana.png', width: 80, height: 80), ], ), )
[Olma va banan rasmlari yonma-yon]
Slayd 17
Rasmlarni ustma-ust

Column widget ichiga rasmlarni qo'yib, ularni vertikal (ustma-ust) joylashtirish mumkin.
Center( child: Column( mainAxisAlignment: MainAxisAlignment.center, children: [ Image.asset('assets/images/sun.png', width: 100, height: 100), SizedBox(height: 20), Image.asset('assets/images/moon.png', width: 100, height: 100), ], ), )
[Quyosh va oy rasmlari ustma-ust]

[2x2 kataklar ichida hayvon rasmlari]
