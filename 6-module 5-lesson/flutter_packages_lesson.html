Slayd 12
HSL Color nima?
HSL = Hue, Saturation, Lightness

Hue (0-360): Asosiy rang (0=qizil, 120=yashil, 240=ko'k)

Saturation (0-1): Rang to'yinganligi (0=kulrang, 1=yorqin)

Lightness (0-1): Yorqinlik (0=qora, 0.5=normal, 1=oq)

Afzallik: Avtomatik chuqur rang hosil qiladi

Slayd 20
Best Practices
Device Preview uchun:
• Faqat debug rejimida ishlating (!kReleaseMode)
• MaterialApp sozlamalarini to'g'ri qiling
• Har xil qurilmalarda test qiling

Pushable Button uchun:
• Elevation 4-12 oralig'ida saqlang
• Muhim tugmalar uchun ishlating
• Matn kontrasti etarli bo'lsin
• Height 50-80 pixel oralig'ida
Slayd 21
Keng tarqalgan xatolar
Device Preview xatolari:
• useInheritedMediaQuery: false qoldirish ❌
• locale va builder qo'shmaslik ❌
• Production'da enabled: true qoldirish ❌

Pushable Button xatolari:
• child emas, text ishlatish ❌
• Juda katta elevation berish ❌
• onPressed null qoldirish ❌
• HSL ranglarni noto'g'ri berish ❌
