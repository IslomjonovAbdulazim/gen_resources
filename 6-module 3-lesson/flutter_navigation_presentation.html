Slayd 9
Navigator.push() nima?
Push: Yangi sahifaga o'tish

Qanday ishlaydi:
• Hozirgi sahifani stack ga qo'yadi
• Yangi sahifani yuqoridan qo'yadi
• Orqaga tugmasini avtomatik qo'shadi

Misol: Telegram'da chat ochganingizda

Slayd 28
Keng tarqalgan xatolar
1. Import unutish:
• Sahifa fayllarini import qilishni unutish

2. Context yo'q:
• Navigator.push() da context bermaslik

3. Required parameter:
• Constructor'da required yozib, ma'lumot bermaslik

4. Pop dan keyin code:
• Navigator.pop() dan keyin kod yozish
Slayd 29
Navigation best practices
1. Named routes ishlatng - katta loyihalarda

2. Ma'lumotni to'g'ri uzating - xavfsizlik uchun

3. Back tugmasini to'g'ri ishlatng - foydalanuvchi uchun

4. Loading state qo'shing - network da ma'lumot olayotganda

5. Error handling - xatolik bo'lganda nima qilish
